## v1.0.0
**发布日期：** 2026年6月28日

**新功能：**
- WebDAV 同步插件，支持多设备剪贴板同步（坚果云、Nextcloud 等）
- 文本翻译插件，支持条目翻译与划词翻译（Microsoft / Google / 百度 / DeepLX / OpenAI 兼容等）
- 插件总开关，未启用时零后台占用
- URL 链接内容类型，自动识别 http(s) / ftp / www 链接并迁移存量记录
- 快速粘贴扩展为 10 个槽位（含数字 0）
- 复制成功反馈与列表入场动画
- 新手引导
- i18n 支持，多语言（简体中文 / English / 繁體中文）
- 设置 → 关于可查看版本更新日志
- 系统托盘图标显示/隐藏

**修复：**
- 从 0.9.x 升级后可能无法启动的问题
- 安装/更新失败及开机自启失效
- 收藏顺序乱跳
- 多屏不同 DPI 下窗口位置偏移、尺寸异常、双屏唤醒首次不居中
- 超大图片导致卡死，新增单张图片大小限制
- 预览窗口偶现无法消失、预览相关设置初始状态不正确
- Explorer 重启后已隐藏的托盘图标重新出现
- 资源管理器重命名时 Win+V 打断输入
- 显示区域提示设置无法持久保存
- 小键盘数字键不支持快捷键
- Win+V 模式下可清空快捷粘贴快捷键
- 拖拽指针样式改为系统 move 光标
- 卡片间距设置无效
- 列表顺序异常跳动问题
- 设置页 Tab 滚动位置未重置、导航指示器不跟随切换
- 失效问题修复
- WebDAV / 翻译插件、管理员模式与同步相关边界问题

**性能优化：**
- 列表滚动与搜索性能优化，修复搜索时偶发白屏
- 搜索关键词上下文与图片加载体验
- 外观主题 Tab 字体列表加载速度

**其他：**
- 移除设置内搜索，调整设置页布局与预览默认值
- 自 v0.9.30 以来首个正式版；插件默认关闭，需手动启用

## v0.4.5
**发布日期：** 2026年2月6日

**新功能：**
- 图片悬浮预览——鼠标悬停缩略图 300ms 弹出独立预览窗口
- Ctrl+滚轮缩放——预览窗口支持平滑缩放（CSS transition，零窗口 resize）
- 缩放百分比显示——缩放时右下角显示百分比，1.2秒自动淡出
- 预览位置偏好——设置中可选自动/左侧/右侧
- 图片预览开关——设置中可启用/禁用悬浮预览
- 缩放步进可调——设置中可调整 5%-50%
- 单图片文件预览——复制图片文件自动显示预览（失败回退为文件卡片）
- GitHub Actions CI——push/PR 自动执行 ESLint + TypeScript + Cargo 检查
- GitHub Actions Release——推送 tag 自动构建

**优化：**
- 图片加载改为 Asset Protocol（替代 base64，零编码开销，浏览器自动缓存）
- 代码模块化重构——ClipboardItemCard 拆分为 3 个文件
- Rust 命令模块化——commands/mod.rs 拆分为 clipboard/settings/file_ops
- 快捷键解析独立——提取 shortcut.rs 模块
- 提取共享辅助函数：with_paused_monitor() 等
- 消除快捷键读取重复——4 处重复代码统一复用 get_current_shortcut()

**依赖变更：**
- 移除 lazy_static，迁移到 std::sync::LazyLock（Rust 1.80+）
- 移除 base64，图片改用 Asset Protocol 加载
- CURRENT_SHORTCUT 改用 parking_lot::RwLock，消除 8 处 .unwrap()

**修复：**
- 移除未使用的 loading 状态（Settings.tsx）
- 空 catch 块改为 console.error 记录错误
- 移除 console.log 调试日志
- 修复 #[allow(dead_code)] 标注在实际使用代码上的问题
- 预览窗口隐藏时清空图片，防止下次显示时闪烁旧图
